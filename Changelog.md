# Changelog

本文件用于维护每个版本对外发布的更新说明。

编写约定：
- 每个版本使用 `## v版本号` 作为标题，例如 `## v0.3.6`

## v0.5.0

### ✨ 新增功能
- 新增手机电脑互传，扫码后可以在电脑和手机之间收发文字、图片和文件
- 新增复制音效设置，复制时可以有声音反馈
- 新增“重置设置”，可以一键恢复默认设置
- 历史记录新增按天保留的控制，方便按自己的使用习惯管理内容
- 设置页中的说明和提示更清晰

### 🚀 优化改进
- 更新提示现在更醒目，检查更新按钮会直接显示在设置页顶部
- 历史记录现在支持分页加载，记录很多时浏览和翻页会更顺手
- 设置页改成了独立页面，并按类别整理，常用设置、历史、快捷键、互传和关于更容易找到
- 剪贴板内容的捕获和复制反馈更稳定，使用时更少遇到内容缺失或反馈不明显的情况
- 设置项修改后会立即生效，不需要额外点保存
- 优化复制速度，提升复制体验

---

## v0.4.1

### 🐛 问题修复
- 修复 macOS 升级后已授权但直接粘贴仍提示缺少“辅助功能 / 自动化”权限的问题，降低自动化脚本失败对直接粘贴的影响。
- 修复 macOS 文本写入剪贴板时的兼容性问题，直接粘贴文本内容更稳定。
- 修复全局快捷键中 `Meta` 修饰键在 macOS / Linux 上显示和保存不一致的问题。
- 修复清空历史记录后仍可能重新捕获同一剪贴板内容的问题。
- 修复面板在显示、隐藏后最小尺寸状态可能丢失的问题。

### 🚀 优化改进
- 优化 Linux 直接粘贴能力检测，在桌面会话识别不明确时可根据已安装工具选择可用后端。

## v0.4.0

### 🚀 优化改进
- 优化 Linux 下历史记录的来源应用识别逻辑。
