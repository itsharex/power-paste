# Changelog

本文件用于维护每个版本对外发布的更新说明。

编写约定：
- 每个版本使用 `## v版本号` 作为标题，例如 `## v0.3.6`
- 标题下方直接填写会展示在应用更新弹窗中的 Markdown 内容
- 使用 `---` 或下一个版本标题结束当前版本说明

## v0.4.0

### ✨ 新增功能
- Linux 直接粘贴新增 Wayland 会话支持；安装 `wtype` 后即可在 Wayland 下直接发送粘贴快捷键，不再仅限于 `X11 + xdotool`。

### 🚀 优化改进
- 优化 Linux 下历史记录的来源应用识别逻辑。
